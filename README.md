📊 Dashboard Vendas - Loja ABC
Este projeto apresenta um conjunto de dashboards interativos para análise de vendas, estoque, clientes e desempenho de funcionários da Loja ABC. O objetivo é fornecer uma visão integrada dos principais indicadores de negócio, facilitando a tomada de decisão e o acompanhamento dos resultados.

🚀 Funcionalidades
Vendas: acompanhamento das vendas por produto, receita faturada e previsão de receita.

Estoque: controle de estoque inicial e final, percentual vendido e status dos produtos.

Clientes: análise de compras por cliente, distribuição geográfica por estado e preferências por tipo de produto.

Funcionários: monitoramento da performance individual dos vendedores em relação às metas estabelecidas.

📂 Estrutura do Projeto
Vendas → Painel principal com resumo de vendas, faturamento e previsão.

Estoque→ Tabela de estoque inicial/final e percentual vendido.

Funcionarios → Comparativo de desempenho dos vendedores com metas.

Clientes → Compras por cliente, estados e produtos.

🛠️ Tecnologias Utilizadas
Power Query para extratificação dos dados.

Power BI para visualização dos dados.

Excel como base de dados inicial.

DAX para cálculos de métricas e indicadores.

📌 Objetivo
O projeto busca oferecer uma visão integrada e estratégica dos principais dados da loja, permitindo:

Identificar produtos com maior e menor desempenho.

Monitorar o cumprimento das metas de vendas por funcionário.

Avaliar a eficiência do estoque.

Entender o perfil e comportamento dos clientes.
